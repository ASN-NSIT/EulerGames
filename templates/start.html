RULES



Welcome to the Euler Games!

1) Your task is to escape out of this labyrinth you find yourself in.
2) The maze consists of several locks ( read levels ), each consisting of a mathematical problem.
3) The solution key to the problem unlocks the next problem.
4) The key to each lock is an integer input (unless stated otherwise in the problem).
5) The answer could be calculated by writing out a simple running code with the appropriate algorithm ( unless of course, you have brains the size of a supercomputer to calculate them manually!)

See you at the end! And oh, the person who escapes out first, wins the game, so no time to lose!
